# Wechat_QQ_wordcloud
## 导出微信或者QQ聊天记录生成词云
### 本文最终目的是将微信/QQ聊天记录从备份文件中提取出来，生成Excel或txt文件，进行文本分析生成好玩的词云图片，得到聊天内容中的热词，后续也可以分析群聊中谁是话痨、词语关联性等。
